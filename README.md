# Expense Tracker with AI module

This project is a web application for tracking personal expenses.
It includes an AI module for automatic expense category classification
based on text description.

## Technologies
- Python
- Flask
- MongoDB
- Hugging Face Transformers
- PyTorch
- Chart.js

## Features
- User authentication
- Expense tracking
- AI-based category suggestion
- Visualization of model training results

## Purpose
Educational project for the course "Information Technologies of Cloud Computing".

## Team workflow
This project uses GitHub branches and pull requests
to organize collaborative development and code review.
